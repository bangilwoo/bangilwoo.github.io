cmd 기본경로(디렉토리) 변경

2021-01-23
2021-02-03
Window10

Window 환경에서 작업을 위해 cmd 창을 이용하는 일이 많이 있습니다. 창을 열게 되면 경로가 C:\WINDOWS\system32> 지정되 항상 경로를 뒤로 돌려야 하는 번거로움이 있습니다.

작업을 좀더 효율적으로 하기위해 시작 경로를 C:로 설정해 봅니다.

cmd 속성 들어가기

  • 설정을 위해 cmd 폴더로 이동합니다.

  • 폴더에 들어가서 cmd 속성창을 열어줍니다.

cmd 시작 경로 변경

  • 설정에 들어가게 되면 바로가기 > 대상에 경로를 확인할 수 있습니다.

  • c: 경로를 설정하기 위해 대상에 /k cd /d C:￦를 입력해 줍니다.
    • 기존 설정 뒤에 추가하면 됩니다.
    • 만약 경로를 추가 하고 싶다면 C:￦뒤에 경로를 설정하면 됩니다.

  • 경로를 입력하고 저장 버튼을 누르고 cmd 관리자 모드로 실행하면 아래와 같이 경로가 바껴 있는 것을 확인할 수 있습니다.
