Linux(CentOS7)로 git(Repositories) 연결하기

git은 소스코드의 효율적인 관리를 위한 형상 관리 도구 입니다.

git을 사용하면 소스코드를 계속해서 주고 받을 필요가 없으며 같은 파일에 대해 여러명이 동시에 작업을 할 수가 있기 때문에 생산성이 매우 용이 합니다.

git 패키지 다운로드

git을 연동 시키기 위해서는 Linux 환경에서 git 패키지를 설치 합니다.

패키지 설치를 위해 yum 명령어를 사용 합니다.

  • sudo는 관리자 명령어를 의미

git 파일 다운로드

패키지를 설치 했다면 사용하고자 하는 git 파일의 주소를 복사하여 터미널에서 실행

  • git파일 다운로드 시 rood계정으로 진행

데이터 로드 확인하기

설치가 완료 되었다면 지정한 위치에 설치가 되었는지 확인 합니다.
