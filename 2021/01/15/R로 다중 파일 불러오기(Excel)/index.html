R, 다중 파일

작업을 진행 하다보면 같은 형식의 파일을 묶어야 되는 상황이 생길때가 많이 있습니다.

같은 형식을 묶어 진행하면 일의 처리 속도가 빨라지게 되므로 알아두면 유용할 것 입니다.

1. 파일 형식 파악하기

  • 파일을 불러오기 전에 형식을 알아보자.

  • 알아본 파일 형식을 기초로 기본 데이터 불러오기

  • 제대로 파일들을 불러왔는지 확인

2. 다중 엑셀 파일 불러오기

  • 파일 목록을 확인
  • 다중 파일 묶기

  • 묶은 파일의 목록을 확인

  • 묶은 파일들을 데이터 형식으로 출력
  • 파일이 제대로 생성 된것을 확인한다.
