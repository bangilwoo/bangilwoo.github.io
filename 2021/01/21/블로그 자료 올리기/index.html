블로그 자료 올리기

2021-01-21
2021-02-03
hexo

블로그를 만들었으면 블로그에 업로드 할 파일을 만들어 봅시다.

경로 설정 하기

  • cmd(명령 프롬프트)를 실행 합니다.

    • 관리자 모드로 실행 합니다.

  • cmd 경로를 현재 블로그 폴더 위치로 지정해 줍니다.

    • 블로그 생성 위치가 다르기 때문에 ex)를 참고 하여 경로를 지정해 줍니다.

    • ex)C:\Users\bangi\Desktop\blog>

      파일 생성 하기

  • 블로그 경로가 지정 되었으면 새파일 생성 명령어를 사용하여 파일을 생성합니다.

    • 생성 명령코드는 3가지 존재 합니다.
      • post: 기본 레이아웃 형식
      • page: 해당 경로 진입시 볼 수 있는 레이아웃 형식
      • draft: 바로 게시하지 않고 초안을 작성하는 형식
        • 작성 완료시 publish 명령어로 배포하여 게시
  • 명령 실행시 파일은 블로그 폴더 내의 source 폴더에 생성 됩니다.

  • source 안에 _posts로 진입하면 {파일명}.md 파일이 존재 합니다.
  • 파일을 실행할 때 연결 프로그램으로 Pycharm 선택하여 작업을 진행 합니다.
  • 진행이 완료 되면 작업한 파일을 저장하고 종료 시킵니다.

파일 업로드

파일 업로드를 위해 파일을 생성하는

  • 작업이 끝나면 hexo 명령을 실행하여 업로드를 진행 합니다.
  • 블로그 생성과 마찬가지로 두개의 명령어를 실행 하여야 배포를 진행 합니다.

    • 위와 같이 명령을 진행 하는 방법도 있지만 시간을 아끼기 위해 아래의 명령을 실행하여 시간을 절약할 수 있습니다.

  • 배포가 끝나게 되면 자신의 블로그로 진입하여 파일이 업로드 되었는지 확인 합니다.
    • 업로드 진행 시 1~5분 정도의 시간이 소요 됩니다.
