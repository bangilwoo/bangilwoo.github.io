Window10

window10은 크게 Retail과 Volume 라이센스로 구분 됩니다. 일단 간단하게 Retail과 Volume 차이에 대해 알아 봅시다.

라이센스 사용처 사용가능 대수
Retail 1PC 개인용 1대 PC 설치 가능
Volume 기업/학교 등 단체용 다수 PC 설치 가능

설치된 window 라이센스 확인하기

  • 좌측 하단 검색창에 windows powershell을 입력합니다.

    • (선택)실행을 관리자 모드로 실행 합니다.

  • powershell 창에 slmgr /dlv입력 합니다.

  • 버전을 확인 합니다.

    • 제품 키 채널에 Retail로 표시 되면 Retail 버전이고 Volume로 표시 되면 Volum 버전이 설치된 것입니다.

      영구적 라이센스 인지 확인하기

  • windows powershell 창을 다시 열어 줍니다.

  • powershell 창에 slmgr -xpr입력 합니다.

  • 영구 라이센스라면 팝업창에 영구적으로 정품 인증 문구가 적혀 있습니다.
