Markdown

Markdown을 사용하여 문법의 작성 및 편집을 유용하게 할 수 있습니다. 이제 Markdown에 대해 알아봅시다.

마크다운(Markdown) 이란?

Markdown은 문서 작성을 지원하는 태그(Tag) 형식의 문법입니다.

Markdown의 장점

  • Web에 적합한 문서 작성 도구
  • 배우기 쉽고 직관적
  • 생산성이 높음
  • HTML로 변환 가능
  • 용량이 적고 보관이 용이
  • git을 통한 버전관리가 가능(Text 저장)
  • Python의 Python의 Jupyter Notebook, R의 R Markdown 등 사용에 용이(Tool)
