USB 초기화 하기

2021-02-22
2021-02-22

USB를 오래 사용하여 속도가 줄어들거나 특정한 파일을 설치(window 체제)하여 USB 메모리의 용량이 줄어드는 경우가 존재 합니다.
이러한 경우 초기화를 통해 대부분의 문제를 해결할 수 있습니다. 핸드폰, 컴퓨터와 같은 경우에는 초기화 툴이 제공 되지만 USB의 경우 툴이 없기 때문에 diskpart 명령어를 통해 초기화를 진행할 수 있습니다.

PowerShell 실행하기

PowerShell 실행을 위해 Win + X → A키를 눌러 주거나 시작 버튼 오른쪽을 클릭하여 실행합니다.

  • PowerShell 관리자로 실행 합니다.
    • Window 명령은 관리자로 실행해야 적용 됩니다.

diskpart 실행하기

diskpart 입력하여 디스크 관리로 진입합니다. 진입후 list disk 명령어로 USB 디스크가 몇 번인지 찾습니다.

  • 보통 제일 처음에 있는 디스크는 메인 디스크(C:)라고 보시면 됩니다.

디스크 선택 및 정리 하기

초기화 하고 싶은 디스크를 select disk 1 명령어로 입력하고 USB 디스크 정리를 위해 clean명령어를 입력합니다.

파티션 설정 및 포맷 하기

기존 파티션이 아닌 새로운 파티션을 만들기 위해 create partition primary 명령어를 입력하고 NTFS 파일 시스템으로 포맷하기 위해 format fs=ntfs quick 명령어를 입력합니다.

만약 NTFS파일이 아닌 다른 파일로 포맷하기 위해서는 ntfs 원하는 파일 형식을 지정해주면 됩니다.
