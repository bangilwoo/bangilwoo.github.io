시작 버튼 오류 해결하기

2021-02-03
2021-02-03
Window10

가끔씩 시작 버튼 명령이 말을 듣지 않을 때가 존재 합니다. 컴퓨터 설정 오류로 인한 문제일 수가 있습니다. 그때 간단한 조치 방법이 존재 합니다.

관리자 및 탐색기 실행하기

작업 관리자 실행

  • 시작 버튼 우클릭 -> 작업관리자 클릭

  • Ctrl + Alt + Del

    파일 탐색기 실행

  • 윈도우 버튼 + E 클릭

  • 아무 폴더 열기 -> 좌측에 즐겨찾기 클릭

관리자를 통한 재시작하기

관리자를 통해 재시작 하기

  • 작업 관리자 창을 열어 파일 탐색기 찾기
    • 작업 프로그램이 많으면 자세히 보기로 찾아보자
  • 파일 탐색기 우클릭하여 다시 시작을 클릭

확인해보기

시작을 눌러서 실행 되는지 확인합니다. 이 방법은 처음 얘기한것 처럼 간단한 조치 방법으로 상황에 따라 개선이 되지 않을 수 있습니다.
